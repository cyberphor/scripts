### A quick "Hello world!" example
1. Use a text-editor to create a source file
2. Add your code to the source file
3. Modify the source file to be executable
4. Run your code

```bash
# step 1
vim ./hello_world.rb
```
```ruby
# step 2
#!/usr/bin/ruby

puts 'Hello world!'
```
```bash
# step 3
chmod +x ./hello_world.rb
```
```bash
# step 4
./hello_world.rb
Hello world! # <--- output
```
